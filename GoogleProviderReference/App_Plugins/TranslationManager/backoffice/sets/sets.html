Translation Sets

Translation Sets define where the content you want to translate comes from, and where you want it to go once it is translated.

A Translation Set has a single master site where all the content in the source language will be stored, and one or more target sites where the translated values will be sent to.

Master Site

The Master Site is the site where the translated content will come from. This will be the master language in all translations.

Target Sites

Target Sites must have a language set in their 'culture and hostnames' section - this is then used as the target language for translations.

Target Sites are automatically related to the master site so that Translation Manager knows how to match the content pages up between the sites.

When you first create your Target Site in Umbraco, it is recommended you do this by making a copy of the Master Site.
