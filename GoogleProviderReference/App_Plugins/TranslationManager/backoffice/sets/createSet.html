Circular Set warning

One or more of your target sets, are the same node as your master.

This can be ok (if you are using things like vorto) but you should exercise caution, and you could translate over the top of the translations if you are not careful.

Make sure you explicity select your vorto aliases in properties box to ensure only vorto type values are translated.
